Track Suggester

About the Track Suggester..

With so many different options to choose from in terms of coding languages, it's easy to see how some students may be overwhelmed at the thought of having to decide on a track after Intro to Programming. With the Track Suggester students can get a suggestion on which track to choose based on there preferences.

Track Suggester Questionaire

Do you want to build interactive web programs on the front end?

Would you want the language you learn to be very simplistic and productive?

Do you want to build back end services but also have the option to build dyvamir web apps?

Would you like to develop Android apps?

Do you want to build backend services with one of the most popular backend languages?

Are you interested in developing backend framework for blogs or corporate, political, and government sites?
